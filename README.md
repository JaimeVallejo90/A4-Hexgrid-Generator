# A4 Hexgrid Generator

A single-page A4 hex grid generator with a focused desktop layout. All measurements are in millimeters and scaled to true A4 size (210 × 297 mm). Designed for GitHub Pages.

## Features
- Hexgrid or hexflower layouts
- True millimeter sizing on an A4 page (portrait or landscape)
- Live preview and export
- Export options:
  - PNG White (keeps colors, white background)
  - PNG Transparent (keeps colors, transparent background)
  - PDF White (keeps colors, white page)

## Run locally
Open `index.html` in your browser.

## Deploy on GitHub Pages
1. Push to GitHub.
2. Enable Pages in repo settings (root or `/docs`).
3. Use the provided URL.

## Notes
- Fonts load after page render; the canvas redraws once fonts are ready.
- Very thin lines may appear lighter on screen but export at full resolution.
